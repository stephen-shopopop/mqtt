# Broker mqtt

## Description

Broker mqtt nodejs - mqtt3.1 [aedes](https://www.npmjs.com/package/aedes)

## Contributing

1. npm run start -  Start broker
2. npm run dev - Start broker with ts-node (dev only)
3. npm run compile - Build ts

## Docs

### Summary

Server start:

- tcp: mqtt://localhost:1883
- ws: ws://localhost:8883
